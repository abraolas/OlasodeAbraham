Microsoft Intune Lab Projects

10 end-to-end labs built as part of my MD-102 Endpoint Administrator certification journey. Each lab reflects real enterprise scenarios across Windows, iOS, and Android device management.

Lab 01
Tenant Setup & Intune Activation
Configured M365 developer tenant, activated Intune, created test users and device groups, and explored the full Intune Admin Centre.
Intune M365 Entra ID
Lab 02
Windows 11 Device Enrollment (Azure AD Join)
Enrolled a Windows 11 VM via Azure AD Join, verified enrollment in Intune portal, and explored remote device management actions.
Windows 11 Azure AD Join MDM Enrollment
Lab 03
Android Device Enrollment (Work Profile & Corporate)
Enrolled Android device using Intune Company Portal, configured Android Enterprise Work Profile for BYOD, and explored Managed Google Play integration.
Android Enterprise BYOD Managed Google Play
Lab 04
iOS/iPadOS Device Enrollment
Set up Apple MDM Push Certificate, enrolled iPhone via Company Portal, and explored supervised vs unsupervised iOS management modes.
iOS/iPadOS Apple MDM Company Portal
Lab 05
Configuration Profiles — Windows & iOS
Built and deployed Configuration Profiles using Settings Catalog for Windows (USB restrictions, screensaver) and Device Restriction profiles for iOS (camera, AirDrop).
Settings Catalog Device Config Policy Assignment
Lab 06
Compliance Policies & Conditional Access
Created compliance policies requiring BitLocker and PIN enforcement, then linked them to Azure AD Conditional Access to block non-compliant devices from Microsoft 365.
Compliance Conditional Access Zero Trust
Lab 07
App Deployment — Win32, iOS & Android
Packaged a Win32 app using IntuneWinAppUtil, deployed to Windows. Deployed managed App Store app to iOS and Managed Google Play app to Android devices.
Win32 App IntuneWinAppUtil App Management
Lab 08
Windows Autopilot Deployment
Extracted hardware hash, registered device with Autopilot, configured user-driven deployment profile, and simulated zero-touch Windows provisioning end-to-end.
Autopilot Zero-Touch OOBE
Lab 09
Security Baselines & Endpoint Security
Applied Microsoft Security Baseline to Windows device, configured Defender Antivirus, Firewall, and BitLocker policies, and reviewed compliance status reports.
Security Baseline Defender BitLocker
Lab 10
RBAC, Scope Tags & Intune Monitoring
Created custom RBAC role with limited permissions, configured Scope Tags to separate device groups by location, and explored audit logs and compliance reports.
RBAC Scope Tags Audit Logs
